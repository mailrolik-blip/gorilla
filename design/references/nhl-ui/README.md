\# NHL UI visual reference



Эта папка содержит визуальные референсы для следующего этапа UI/UX-пересборки Gorilla.



\## Файлы



\- `01-dark-hub.jpg`

\- `02-versus.jpg`

\## Что нужно взять из референсов



Не копировать экран один в один.  

Использовать как mood / system reference.



Основные ориентиры:



\- тёмный общий фон как базовая продуктовая среда

\- ощущение хоккейной арены / game shell

\- крупная и уверенная типографика

\- один главный focal area на экран

\- меньше карточек и коробок

\- меньше дробления на плашки

\- больше воздуха

\- cleaner hierarchy

\- высокий контраст

\- акценты в amber / gold

\- image-led композиция

\- ощущение современного спортивного digital product, а не CRM из прямоугольников



\## Что особенно важно для Gorilla



Нужно уйти от:

\- бесконечных карточек

\- box-zoning под каждым элементом

\- случайных прямоугольников

\- сжатых секций без воздуха

\- одинаковой визуальной важности всех блоков



Нужно прийти к:

\- одному цельному dark product shell

\- более кинематографичной и спортивной подаче

\- сильной главной странице с крупными image-блоками

\- более живому /cabinet и /admin

\- cleaner workspace-подходу

\- логичному масштабу и иерархии



\## Что запрещено



\- не копировать NHL pixel-perfect

\- не использовать чужие логотипы, шрифты, персонажей и ассеты

\- не делать светлую основу

\- не возвращать интерфейс к набору карточек в карточках

\- не делать просто ещё одну перестановку существующих блоков



\## Цель



Сделать Gorilla самостоятельным продуктом в духе современного NHL game UI:

тёмный, сильный, спортивный, с воздухом, крупной типографикой и одной понятной рабочей средой.
