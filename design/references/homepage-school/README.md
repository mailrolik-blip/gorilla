# Homepage School References — Gorilla Hockey

Эта папка содержит визуальные и смысловые референсы для новой главной страницы хоккейной школы Gorilla Hockey.

## Файлы

- `01-gorilla-logo-reference.png`
  - логотип школы
  - использовать как бренд-опору в хедере, hero и подвале
  - не искажать, не перерисовывать

- `02-school-promo-poster-reference.png`
  - референс по духу школьного хоккейного промо
  - важны:
    - энергия
    - спортивная подача
    - крупный контрастный текст
    - ощущение живой школы
  - не копировать постер буквально

- `03-nhl-dark-shell-reference.png`
  - основной референс по UI grammar
  - брать:
    - dark shell
    - крупную типографику
    - сильную focal area
    - ритм секций
    - композицию
    - спортивную digital-подачу
  - не использовать как literal image на странице

- `04-trainers-card-reference.png`
  - референс для блока тренеров
  - брать:
    - структуру карточки
    - читаемость
    - спортивную подачу
  - не копировать один в один

## Что нужно взять из референсов

- тёмную спортивную среду
- сильный hero-block
- крупный заголовок
- визуальные зоны с фото / иллюстрациями
- понятные CTA
- меньше мелких рамок и карточек
- больше воздуха
- меньше текста на экране
- secondary info скрывать под hover / tooltip / accordion / expandable blocks

## Что нельзя делать

- не копировать NHL 1:1
- не использовать чужие бренды как часть финального дизайна
- не строить всю страницу из одинаковых прямоугольных карточек
- не делать длинные текстовые полотна
- не хардкодить всё в app/page.tsx одной простынёй

## Цель

Собрать новую главную страницу Gorilla Hockey как современную спортивную landing/home страницу:
- школа
- команды
- тренировки
- галерея
- тренеры
- аренда
- отзывы
- миниигра
- карта
- контакты

Страница должна ощущаться как сайт живой хоккейной школы, а не как CRM или набор тёмных блоков.
